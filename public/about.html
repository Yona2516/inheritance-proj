About the System

This Inheritance Management System is designed to securely manage and store will documents and beneficiary details. It allows administrators to register beneficiaries, upload will documents in PDF format, and manage beneficiary data.

The main features of the system include:

  • Registering beneficiaries with personal and contact information
  • Uploading and storing will documents in PDF format
  • Viewing a list of all registered beneficiaries
  • Editing or deleting beneficiary records
  • Sending SMS notifications (via Beem integration)

The system ensures that data is stored safely and provides a user-friendly interface for managing inheritance-related information. It is built using Node.js for the server, MySQL for the database, and plain HTML/CSS for the frontend.

For more details or technical support, please contact the system developer or administrator.
